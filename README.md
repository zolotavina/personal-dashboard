# Personal Dashboard

A clean personal productivity dashboard built with Next.js, React, TypeScript and Tailwind CSS.

## Purpose

The goal of this app is to create one place for daily organization, self-management and reflection.

## Planned Features

- Task list
- Notes section
- Daily habits / routines tracker
- Mood tracker
- Energy tracker
- Persistent saved data
- Clean responsive UI

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Local Storage (initial version)
